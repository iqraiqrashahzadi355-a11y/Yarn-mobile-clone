# Yarn Landing Page Clone 🚀

A responsive static landing page inspired by the Yarn website.
Built using **HTML5** and **CSS3 (Flexbox layout)**.

---

## 🌐 Live Preview

You can host this project using:

* GitHub Pages
* Netlify
* Vercel

---

## 📌 Features

✔ Top Announcement Bar

✔ Responsive Navigation Header

✔ Search Section

✔ Hero Section with Call-To-Action

✔ Features Section with Image

✔ Extra Features Grid (3 items per row using Flexbox)

✔ CTA Section

✔ Footer

---

## 🛠 Technologies Used
